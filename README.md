# Proyecto de Análisis de Datos – Energía y Transición

## 📌 Descripción
Este proyecto forma parte del curso de análisis de datos y tiene como objetivo explorar, limpiar, transformar y visualizar información relacionada con la transición energética.  
Se incluyen scripts, notebooks y documentación que permiten reproducir el flujo completo de análisis.
